# Smart-Audio-Focus

**「今、聴きたい音だけを。」** 複数のタブで動画や音楽が流れてしまうストレスを解消し、ブラウジングの集中力を最大化するChrome拡張機能です。

---

## 📺 デモ (ここに動画を貼り付け)
## ✨ 主な機能
* **オートフォーカス・ミュート**: 現在開いているタブでメディアが再生されると、他のタブの音を自動でミュートまたは一時停止します。
* **バックグラウンド制御**: タブを切り替えた際、以前のタブの音声を自動的に制御し、混線（カオス状態）を防ぎます。
* **シンプル設計**: 設定不要で、インストールした瞬間から「静かなブラウジング」を実現します。

## 🛠 使用技術
* **Language**: JavaScript (Content Scripts)
* **API**: `chrome.tabs`, `chrome.runtime`
* **Target**: Google Chrome / Microsoft Edge

## 📦 インストール方法
1. このリポジトリをクローンするか、ZIPでダウンロードします。
2. Chromeの拡張機能管理画面 (`chrome://extensions/`) を開きます。
3. 「デベロッパー モード」をオンにします。
4. 「展開して読み込む」をクリックし、本プロジェクトのフォルダを選択してください。

---

## 💡 開発の背景
「YouTubeを見ている最中に別のタブから広告が流れてくる」「作業中に複数の音が混じって集中できない」という、インドア派なら誰もが抱えるストレスを解消するために開発しました。
